# Actividad-Fundamental-N.-3-Producto-Integrador-de-Aprendizaje

# Predicción de Costos Médicos con Regresión Lineal

Este repositorio contiene la implementación de un modelo de aprendizaje supervisado (**Regresión Lineal**) para predecir costos médicos de seguros basándose en variables como edad, índice de masa corporal y tabaquismo.

## Descripción de la Actividad

El objetivo es demostrar la carga de datos, el uso de técnicas de preprocesamiento (Normalización y Encoding) y la evaluación de modelos de regresión en Python.

## Tecnologías utilizadas

* Python 3
* Scikit-learn (Modelo y Preprocesamiento)
* Pandas (Manipulación de datos)
* Matplotlib & Seaborn (Visualización)

## Cómo ejecutar el código

1. Clonar el repositorio o descargar el archivo `Actividad_fundamental_3.py`.
2. Instalar las dependencias:

```bash
pip install pandas matplotlib seaborn scikit-learn
```

## Ejecucion
```bash
python Actividad_fundamental_3.py
```

## Resultados

Al ejecutar el código se mostrarán en consola las métricas de desempeño ($R^2$, MSE) y se generará una gráfica de dispersión comparando los valores reales vs. los predichos.
